Detailed Description

Outlier detection (using median absolute deviation)

Note
D - shared3p protection domain
Supported types - int32 / int64
Constant 1.0 is used as the parameter for median absolute deviation.
Parameters
data- input vector
isAvailable- vector indicating which elements of the input vector are available
lambda- constant. The value of lambda depends on the dataset. Anything from 3 to 5 can be used as a starting value.
Returns
returns a boolean mask vector. For each sample point x, the corresponding mask element is true if the corresponding isAvailable element is true and its absolute deviation from the median of the sample does not exceed lambda Â· MAD where MAD is the median absolute deviation of the sample.
Leakage
None
