Sharemind SDK Beta

Introducing the Sharemind Software Development Kit

Sharemind is a novel database and application server that can process encrypted data without decrypting it. Its is based on secure multi-party computation and has been used to protect privacy and business secrets in various applications.

The experimental Sharemind SDK contains the SecreC 2 programming language that separates public data and secrets on a type system level and an emulator that you can use to estimate the running time of your application in a fully encrypted environment. Your SecreC programs will be fully compatible with the Sharemind Application Server, that runs the code using full cryptographic protection and provides support for enterprise applications.

Download version 2015.01

There are two ways for getting the SDK.

  1. Download a preinstalled virtual machine image from http://sharemind.cyber.ee/sdk.html. This can be used with a tool like VirtualBox.
  2. Build the tools from source code at https://github.com/sharemind-sdk/build-sdk.

Getting started

The following assumes that you have installed the virtual machine image into a tool like VirtualBox.

  1. Run the Qt Creator tool from the desktop.
  2. Open an example program from /home/sharemind/demos with Ctrl+O (or select File->Open File or Project... from the menu).
  3. Compile the program with Ctrl+F1 (or select Tools->External->Sharemind SDK->Compile SecreC from the menu).
  4. Run the program in the emulator with Ctrl+F2 (or select Tools->External->Sharemind SDK->Run SecreC from the menu).
  5. Read the printed output and the estimated running time from the console window.

Known issues

The SDK is limited to developing and profiling single secure programs. To license the full Sharemind system, please contact Cybernetica.

These issues affect version 2015.01:

  • This version does not yet have any database support.
  • The performance models currently represent a deployment with direct 1 Gbit network connections between the servers. Future versions will be more configurable.
  • Not all private operations have performance models. However, most of the arithmetic is modelled well.

Please report other issues at https://github.com/sharemind-sdk/build-sdk/issues.
