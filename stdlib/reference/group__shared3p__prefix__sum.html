Detailed Description

Fast prefix sum.

Note
D - shared3p protection domain
Supported types - uint8 / uint16 / uint32 / uint / int8 / int16 / int32 / int
Parameters
vec- a vector of supported type
Returns
Turns the input vector [x1, x2, ..., xn] to [x1, x1 + x2, ... , x1 + ... + xn]
