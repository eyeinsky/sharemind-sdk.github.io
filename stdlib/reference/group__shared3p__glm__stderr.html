Detailed Description

Estimate the standard errors of coefficients of generalized linear models.

Note
D - shared3p protection domain
Supported types - int32 / int64 / float32 / float64
Parameters
dependent- sample vector of dependent variable
variables- a matrix where each column is a sample of an explanatory variable
coefficients- coefficients estimated by the GLM fitting procedure
family- indicates the distribution of the dependent variable
Returns
returns a vector with the standard errors of each coefficient
Leakage
None
