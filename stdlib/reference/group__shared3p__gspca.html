Detailed Description

Principle component analysis. Note that this method is relatively efficient and precise when a low number of components is required. It uses fixed point numbers internally so it may fail on large inputs due to overflow.

Note
D - shared3p protection domain
Supported types - float32 / float64
Parameters
X- data matrix where the columns are variables
n_components- how many components to compute
iterations- how many iterations to run the algorithm
returnValues- indicates which results to return. Use bitwise or if you want multiple results.
Returns
PCAResult structure
Leakage
None
