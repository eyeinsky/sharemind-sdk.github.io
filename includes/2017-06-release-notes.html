Compatibility notes

This version of the Sharemind SDK is not compatible with older versions of the Sharemind SDK, including on the network protocols, configuration and API/ABI levels.

Features

Emulator

  • Sharemind components no longer search for their configuration from the shell's working directory. Sharemind Emulator looks for its configuration according to the XDG Base Directory Specification, i.e. from $XDG_CONFIG_HOME/sharemind/emulator.conf, where $XDG_CONFIG_HOME defaults to $HOME/.config:/etc/xdg if it is not defined in the environment. Sharemind components support the --conf command-line parameter override these defaults.
  • Files referenced in Sharemind configuration files are loaded relative the shell's working directory. Use the %{CurrentFileDirectory} placeholder that is replaced with the absolute path of the current configuration file.
  • Sharemind Emulator binary has been renamed: Emulator -> sharemind-emulator

SecreC standard library

  • Added isSuffixOf function to SecreC standard library.
